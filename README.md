# AI Review Replier

Editable SOUS Review Reply Agent prototype.

## Run locally

```bash
npm run dev
```

Then open <http://localhost:4174/Review%20Reply%20Agent.html>.

## Main files

- `Review Reply Agent.html` wires the page together.
- `styles.css` contains the visual styling.
- `app.jsx`, `panels.jsx`, `data.jsx`, and `icons.jsx` contain the React prototype.
- `_ds/sous-design-system-26-019e0688-cda5-774f-bed0-d9a8f61ab6f2/` is the bundled SOUS design system snapshot included with this export.
